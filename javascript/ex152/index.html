Variavéis - Introdução

O que são variáveis?

Tipos de variáveis

  • String
  • Number ..... subtítulo int/Float
  • boolean ..... true/false

Regras ára declaração de variáveis

  • Não podem ser iniciadas com números, apenas com letras ou "_"
  • Não podem ser utilizados caracteres especiais como "ç", "^","~"
  • Não podem ser iguais as palavras reservadas a linguagem

    • Declando variáveis com a instrução "var"

    Case sensitive

    var curso, var Curso e var CURSO são variaéveis diferentes

    Exemplos

    var 123nome, var número
